# 🛡️ IntelliCredit: The Neuro-Shield Hackathon Project

Welcome to the **IntelliCredit** repository! This application was developed for **Hack For Green Bharat: The Intelli-Credit Challenge**. 

This project is a modern, AI-powered credit analysis tool designed to assess credit risk through advanced heuristics and a responsive user interface.

## 🚀 Overview

IntelliCredit provides instant, local heuristic analysis through its "Neuro-Shield" engine, offering a unique approach to credit scoring and risk assessment. The application features a clean, responsive UI built with React and a robust Node.js backend.

## 💻 Technology Stack

*   **Frontend**: React 19 (Vite), Framer Motion, Lucide React
*   **Backend**: Node.js, Express.js (REST API concept)
*   **Styling**: Modern CSS with smooth gradients and micro-animations

## 🛠️ Getting Started

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18 or higher recommended)
*   npm (comes with Node.js)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Parshav2304/IntelliCredit.git
    cd IntelliCredit
    ```

2.  **Setup the Frontend:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
    *The frontend will start running, usually at `http://localhost:5173`.*

3.  **Setup the Backend:**
    ```bash
    # Open a new terminal window/tab
    cd backend
    npm install
    # (Assuming you add a start script in the future, e.g., node index.js)
    ```
